# Thread Rose Co. Website

## Description
This is an e-commerce website for Thread Rose Co., a company that creates and sells modern handmade embroidery.

The site will feature a landing page, shop page listing all available products for sale, individual product pages for a more detailed view of individual products, a shopping cart, checkout and payment functionality (Stripe).

The site owner will have an admin account and dashboard page to add and remove products as necessary.

Reports will be generated for the site owner with sales and other statistics.

## Technologies
### Frontend
* React + TypeScript + Tailwind CSS

### Backend
* Node.js / Express

### Database
* PostgreSQL

### Other
* Docker
* Postman
